# voxify
Post Analysis

This script performs exploratory data analysis and model training on a dataset of product reviews. It loads the data, performs initial cleaning and feature engineering, including text extraction and wordcloud generation. Finally, it trains and evaluates a Multinomial Naive Bayes classifier for sentiment prediction. The code emphasizes data preprocessing and model evaluation for insights into the review data. It utilizes techniques such as CountVectorizer and Multinomial Naive Bayes for feature extraction and classification.
